# Holly Lombardo — Personal Brand Portfolio

A personal brand resume website showcasing marketing, 
content strategy, and consulting work across Africa, Asia, 
and Australia.

## Built with
- HTML, CSS, Vanilla JavaScript
- D3.js (interactive world map)
- Tabler Icons

## Features
- Interactive geographic portfolio map
- Filterable skills showcase (30 skills)
- Scroll-triggered animations
- Fully responsive
